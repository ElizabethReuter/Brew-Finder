About the Creators

Owen Roth, Elizabeth Reuter, and Trey Jahner are three up and coming coders who are learning the basics. They recently were assigned a project where they needed to make a functional website that has some practical use! They decided to make a brewery finder, which the user inputs any US city and the application outputs a map of the city with a list of all the breweries in the area.


Challenges Experienced

The challeneges that came up during this project dealt with the use of the map, search function and display of breweries. The map needed to be adjusted to fit in the container and continue to show each city. The search function had issues pulling information when doing a second search. The display of the breweries extended outside of the container if the list exceed more than 12.


Api's used -

  • BreweryDB
  • MapQuest
